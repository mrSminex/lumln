# LUM'N — Telegram-бот для парфюмерной студии


## Стек

| Компонент | Технология |
|-----------|-----------|
| Бот | Python 3.11+ · aiogram 3.x |
| База данных | SQLite (файл `lumln.db`) |
| Веб-панель | FastAPI + Jinja2 |
| Запуск | один процесс, `python run.py` |

---

## Установка и запуск

### 1. Установить Python 3.11+

Скачать: https://www.python.org/downloads/

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Настроить конфиг

Скопировать файл `.env.example` → `.env` и заполнить:

```bash
cp .env.example .env
```

Что нужно заполнить в `.env`:

| Параметр | Как получить |
|----------|-------------|
| `BOT_TOKEN` | Написать @BotFather → /newbot → скопировать токен |
| `ADMIN_IDS` | Написать @userinfobot — покажет ваш Telegram ID |
| `ADMIN_PASSWORD` | Придумать пароль для веб-панели |
| `YCLIENTS_URL` | Ссылка на запись в вашем YCLIENTS |
| `PRIVACY_URL` | Ссылка на политику конфиденциальности (можно разместить на telegra.ph) |
| `OFFER_URL` | Ссылка на публичную оферту |

### 4. Создать QR-код для стола

Ссылка для QR-кода:
```
https://t.me/ИМЯ_ВАШЕГО_БОТА
```
(имя бота — то, что вы указали при создании у @BotFather)

### 5. Запустить

```bash
python run.py
```

После запуска:
- **Бот** работает в Telegram
- **Веб-панель** доступна на `http://localhost:8080`

## Как пользоваться

### Для гостя (сценарий за столом)

1. Сканирует QR → открывает бота
2. Вводит имя и телефон
3. Даёт согласие на обработку данных
4. Получает регистрацию — бот запомнил его

После сессии мастер добавляет формулу → гость получает уведомление в бот.  
В любой момент гость может открыть бота и нажать **«Мои формулы»**.

### Для сотрудника (веб-панель)

Открыть в браузере: `http://localhost:8080`  
Пароль — из файла `.env` (параметр `ADMIN_PASSWORD`)

**Добавить формулу клиенту:**
1. Открыть **Клиенты**
2. Найти нужного (поиск по имени/телефону)
3. Открыть → раздел «Добавить новую формулу»
4. Ввести название и состав → Сохранить

Клиент сразу получит уведомление в Telegram.

**Рассылка:**  
Раздел **Рассылка** → ввести текст → Отправить.  
Поддерживается `<b>жирный</b>` и `<i>курсив</i>` (HTML Telegram).

### Для администратора (через бот)

Если ваш Telegram ID указан в `ADMIN_IDS`, напишите боту `/admin`.  
Доступны: список клиентов, добавление формул, рассылка.

## Компании (сценарий «пришли вместе»)

Каждый гость сканирует QR со своего телефона → регистрируется отдельно → получает свою формулу.  
Никакой привязки к «организатору брони» нет — каждый становится независимым клиентом в базе.

## База данных

Файл `lumln.db` — SQLite, создаётся автоматически при первом запуске.

**Резервная копия:** просто скопируйте файл `lumln.db`.  
Рекомендуется делать копию раз в неделю (или настроить автокопирование).

Таблицы:
- `clients` — клиенты (id, telegram_id, имя, телефон, дата)
- `formulas` — формулы (id, client_id, название, состав, дата, кто добавил)
- `broadcasts` — история рассылок

## Деплой на сервер (когда нужен постоянный запуск)

Для работы 24/7 нужен VPS-сервер. Рекомендуемый вариант — **Timeweb Cloud** или **Selectel**, тариф от 200₽/мес.

Запуск через systemd (Linux):

```ini
# /etc/systemd/system/lumln.service
[Unit]
Description=LUMLN Bot
After=network.target

[Service]
WorkingDirectory=/opt/lumln
ExecStart=/usr/bin/python3 run.py
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
```

```bash
systemctl enable lumln
systemctl start lumln
```
